# Tic-Tac-Toe Game

## About the app

This application features a virtual tic-tac-toe game that a user can play against themselves. A user will be required to sign up, then sign in in order to play. They will also be able to sign out when they are finished playing. Additionally, they have the opportunity to start a new game when their game is over. 

## User Stories

- It is easy & clear to see how to sign up, sign in, and sign out
- It is easy to understand how to play the game
- The game experience is just as good on mobile as on desktop
- It is clear to me when the game is over
- It is clear to me who won the game
- The game is visually appealing
